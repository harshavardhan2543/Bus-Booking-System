# Bus Booking System

## 📌 Description
This is a Java-based Bus Booking System developed by a team of 4 members.

## 🚀 Features
- User Registration & Login
- Search Buses
- Seat Booking
- Ticket Management

## 🛠️ Technologies Used
- Java
- JSP & Servlets
- HTML, CSS
- Maven
